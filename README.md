## üöÄ Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software or dependencies required

### Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/network-packet-analyzer.git
```

Navigate to the project directory:

```bash
cd network-packet-analyzer
```

Open the HTML file in your web browser:

```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

### Usage

**Start Packet Capture**

- Click the "Start Capture" button to begin packet simulation
- Watch as packets appear in real-time

**Apply Filters**

- Use the filter controls to narrow down packet display
- Filter by protocol, IP address, or port number

**Inspect Packets**

- Click on any packet row to view detailed information
- Examine packet headers, payload data, and metadata

**Monitor Statistics**

- View real-time statistics in the control panel
- Track packet counts and data transfer amounts
